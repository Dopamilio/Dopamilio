# Dopamilio — Project Context

## What this is
Dopamilio is an OP-721 NFT collection on Bitcoin L1 (OPNet).
3,333 supply. Mint price: 6,969 sats. Max 5 per wallet (WL: 3).
Live at: https://dopamilio.xyz

## Folder structure
```
dopamilio/
├── index.html               ← the entire site (HTML/CSS/JS, single file ~500KB)
├── vercel.json              ← Vercel config + CSP headers
├── gallery-lore.js          ← lore engine (loaded by index.html)
├── gallery-card.js          ← card terminal renderer (loaded by index.html)
├── mainnet.md               ← mainnet switch guide — READ BEFORE DEPLOYING
├── scripts/
│   └── switch-to-mainnet.sh ← one-command mainnet config switch
├── api/
│   ├── nft.js               ← metadata serverless endpoint /api/metadata/:id
│   ├── quotes.js            ← livebar market data (Twelve Data API)
│   ├── proof.js             ← WL merkle proof endpoint /api/proof?addr=
│   └── attrs.json           ← compact attributes for all 3333 NFTs
├── memes_mlg/               ← MLG webp sticker images + audio
├── nft_preview/             ← preview webp images (sample only, not full collection)
├── fonts/                   ← woff2 font files
├── contracts/               ← AssemblyScript OP-721 contract
│   ├── op721/DopamilioNFT.ts   ← contract source
│   ├── build/               ← compiled DopamilioNFT.wasm (do not touch)
│   ├── abis/                ← ABI JSON/TS files
│   ├── deploy.ts            ← deploy script
│   ├── init-wl.ts           ← initialize WL phase on-chain
│   ├── generate-merkle.ts   ← generate WL merkle tree from wl.txt
│   ├── setup-v2.ts          ← post-deploy verification
│   ├── update-onchain.ts    ← update on-chain metadata (baseURI etc)
│   ├── diagnose.ts          ← diagnostic utility (query contract state)
│   ├── wl.txt               ← whitelist addresses (MLDSA hex or opt1p)
│   └── wl-root.txt          ← Merkle root (generated by generate-merkle.ts)
└── progress_images.json     ← Pinata CID map for all 3333 tokens
```

## Contract deploy workflow
```bash
cd contracts
npm run build && npm run verify   # must print schnorr_gone=true chainErr_gone=true
OPNET_MNEMONIC="..." npx tsx deploy.ts
OPNET_MNEMONIC="..." npx tsx setup-v2.ts <addr>
# Then run: bash scripts/switch-to-mainnet.sh <addr> <p2tr> <block>
# Then: git add index.html vercel.json && git push
```

## Key JS functions in index.html
- `_dopaGet(id)` → returns {rank, tier, tierIdx, traits, traitPct} for NFT id
- `openNftDetail(id)` → opens the gallery card modal
- `_setSoldOut(flag)` → controls sold-out overlay
- `updateProg(n)` → updates mint progress bar
- `_hexToOpnet(hex)` → converts 0x MLDSA identity → opt1... bech32m address
- `galleryRender()` → renders current page of gallery (debounced 30ms)

## Metadata
- Images: Pinata IPFS — https://chocolate-decisive-silverfish-606.mypinata.cloud/ipfs/{CID}
